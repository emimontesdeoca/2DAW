3.- Las variables "max-age", "path" y "domain" son nombres especiales de variables que son reconocidas por un navegador para controlar diferentes aspectos. Investiga que significa cada una de esas variables y explica su significado y uso en el propio fichero.

domain: devuelve el nombre de dominio de la pagina actual // Ejemplo de uso: var x = document.domain; => www.w3schools.com

path: pagina donde se guarda la cookie en el servidor.

Ejercicio 5

5.- Añade un botón que lo que haga es eliminar la cookie del apartado 2. ¿Cómo comprobarías que se ha eliminado? (La respuesta ponla en el código como un comentario)

Solo hay que redefinir la cookie y asignarle un max-age de 1 segundo y se borrara, para comprobarlo solo hay que ver si al utilizar el metodo getCookie devuelve un valor null.

Codigo para la comprobación:

function act5() {

  var cookie = getCookie("color");

  if (cookie == null) {
    alert("La cookie no existe!");
  } else {
    setCookieMaxAge("color", "red", 1);
    alert("Cookie actualizada con 1 segundo de vida!");
  }
}
