La usabilidad es la facilidad con las que las personas interactúan con una herramienta con el fin de alcanzar un objetivo concreto. Lo que nos quiere decir, es que tenemos que facilitar el camino al usuario que usa nuestra web para que no se sienta frustrado por no conseguir lo que busca. El diseño UX es en lo que se basan la mayoría de los diseñadores web para realizar sus proyectos. El Diseño Centrado en el Usuario(DCU) no es otra que diseñar pensando en mejorar la usabilidad, la accesibilidad y la satisfacción que obtienes al interactuar con una interface.
