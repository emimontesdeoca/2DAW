Ejercicio 1

Realizar un programa que pida al usuario mediante prompt 6 valores enteros que serán almacenados en 6 variables distintas. Una vez introducidos se deberá mostrar un mensaje que indique el número mayor, el menor, la suma de los enteros introducidos y la diferencia entre el mayor y el menor.

Ejercicio 2

Solicitar al usuario un número entero a través de una ventana de diálogo. Multiplicar ese número por 2.5 y sumarle 100. Mostrar el resultado en otra ventana de diálogo.

Ejercicio 3

Repetir el ejercicio 1 con cuatro valores enteros y tres valores reales. En este caso, los números deben ser almacenados en una única variable, deben ser todos positivos y se debe mostrar un mensaje de “Aprobado” cuando la media supere el 5. Si la media es menor a 5 saldrá el mensaje "Ponte las pilas"

Ejercicio 4

Se deben añadir 5 imágenes a la página web para que al hacer click sobre ellas se muestre un mensaje como el siguiente: “Haz hecho click sobre la imagen número 1”, “Haz hecho click sobre la imagen número 2”, etc

Ejercicio 5

Solicitar por teclado mediante prompt un número positivo y mostrar un mensaje indicando el número de dígitos que contiene y la suma de cada uno de sus dígitos. Por ejemplo: 125, tendría tres dígitos y la suma de estos sería 8

Ejercicio 6

Solicitar que el usuario escriba alguna de estas cuatro palabras (profesor, manzana, servidor, mar) para luego mostrar la palabra traducida en inglés. Es decir, si se ingresa 'mar' debemos mostrar el texto 'sea' en la página. Solo se admiten estas palabras como entradas permitidas.

Ejercicio 7

Solicitar al usuario el día, mes y año de nacimiento. En función de estos datos mostrar al usuario su edad.

Ejercicio 9

Crear una función que reciba un caracter (por ejemplo el *) y un número. Esta función debe escribir un triángulo formado por esa letra, que tenga como anchura inicial la que se ha indicado en el número.

Ejercicio 10

Modificar el ejercicio anterior para pedir dos caracteres en lugar de uno, se seguirá pidiendo la anchura pero se aplicará un estilo diferente a cada carácter, de forma que sea fácil su distinción visual. Para ello se añadirán estilos CSS que deberán estar separados del código HTML y JS.
