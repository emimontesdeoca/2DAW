Me gusta:

Realizar una versión simplificada del juego "Ahora Caigo" de TV consistente en:

  • que al usuario le aparecerá una pregunta por pantalla. También le aparecerá la respuesta y algunas letras estarán substituidas por asteriscos (por ejemplo, a la pregunta: "Archipiélago conocido como 'Las afortunadas'" se le presentaría una respuesta (que no es única, depende de tu criterio, a más , mayor dificultad): I*AS **N*R*AS
  • El contenido de los asteriscos será editable, o sea, seleccionando el área con el ratón se podrá escribir y eliminar el texto de ese área.
  • Una vez esté completada la respuesta, se pinchará en el botón de “Comprobar”. El script comprobará si está la respuesta es correcta o no, avisando del resultado.
  • Este script irá guardando un histórico de todos los intentos, de forma que se mostrarán, en la misma pantalla, los textos que fueron enviados a comprobación. Para almacenar estos textos utilizaremos localStorage. Una vez terminado el script, deberemos hacer varias pruebas, unas poniendo bien la respuesta y otras fallando.
  • Cuando tengamos el historial bastante lleno, cerraremos el navegador. Una vez cerrado lo volveremos a abrir y comprobaremos que se mantiene el historial.
  • También se incorporará un botón de “Reiniciar” para eliminar nuestro historial. Mostrar al profesor en el aula para su evaluación.
